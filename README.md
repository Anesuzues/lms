# NexaTalent Launchpad - Workplace Readiness Course

A free online course platform designed to prepare students for workplace success through AI skills, CV preparation, interview readiness, and professional conduct training.

## 🚀 Live Demo

**Frontend**: https://limpopo-launchpad.vercel.app  
**Course Platform**: Fully functional with Google Apps Script backend

## 📚 Course Modules

- **Module 0**: AI Skills & Job Search Readiness
- **Module 1**: CV Preparation with AI
- **Module 2**: Interview Readiness with AI  
- **Module 3**: AI for Work & Productivity

## 🛠️ Technologies Used

- **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Google Apps Script
- **Database**: Google Sheets
- **Email**: Gmail API
- **Deployment**: Vercel (Frontend), Google Apps Script (Backend)

## 🏗️ Project Structure

## 🚀 Quick Start

### Prerequisites
- Node.js & npm installed
- Google account for backend setup

### Frontend Development

```bash
# Clone the repository
git clone https://github.com/Anesuzues/limpopo-launchpad.git

# Navigate to project directory
cd limpopo-launchpad

# Install dependencies
npm install

# Start development server
npm run dev
```

### Backend Setup

1. Follow the `GOOGLE_APPS_SCRIPT_DEPLOYMENT.md` guide
2. Set up Google Forms for each module
3. Deploy Google Apps Script backend
4. Configure email notifications
